# Daam Academy

## Setup
1. `npm install`
2. `.env.example`-ийг `.env.local` болгож rename хийнэ
3. Supabase project үүсгээд env-үүдээ бөглөнө
4. `npm run dev`

## Local URL
- http://localhost:3000

## Supabase SQL
Supabase SQL Editor дээр `supabase-schema.sql` файлыг ажиллуул.
